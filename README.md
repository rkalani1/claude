# Claude

A standalone, beginner-first command center for using Claude across chat, Projects, Artifacts, Office, Chrome, desktop, mobile, Claude Code, prompt optimization, agentic workflows, and official feature updates.

Live site: <https://rkalani1.github.io/claude/>

## Develop

This is a static site. Open `index.html` directly or serve the folder locally:

```sh
python3 -m http.server 8129
```

## Verify

- Check source links in the Sources section.
- Run a browser accessibility pass on desktop and mobile.
- Confirm path picker, task picker, prompt lab, answer fixer, reliability prompt, agent workflow prompt, feature posts, disclosure panels, and prompt copy buttons work.
